# TEST PORTFOLIO

## HOW TO USE

* Get the latest version of the PHP starter : 
    
    ```bash
    git clone git@gitlab.com:niwee/starters/starter-php.git portfolio
    ```

* Step into the directory :
    
    ```bash
    cd portfolio
    ```

* Get this project :
    * Use the dedicated command :
    
        ```bash
        make clone
        ```
    * When prompted, enter :

        ```bash
        git@gitlab.com:nicolas-boyer/portfolio_v2.git
        ```

* Launch the Project :
    ```bash
    make all
    ```

* Import the database : 
    ```bash
    make dbimport
    ```

* For more info : 
    ```bash
    make help
    ```
